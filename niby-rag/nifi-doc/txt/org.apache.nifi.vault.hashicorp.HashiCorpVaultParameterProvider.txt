 Bundle org.apache.nifi | nifi-hashicorp-vault-nar Description Provides parameters from HashiCorp Vault Key/Value Version 1 and Version 2 Secrets. Each Secret represents a parameter group, which will map to a Parameter Context. The keys and values in the Secret map to Parameters.
