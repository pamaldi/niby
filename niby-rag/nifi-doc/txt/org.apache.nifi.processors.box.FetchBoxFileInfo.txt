 Bundle org.apache.nifi | nifi-box-nar Description Fetches metadata for files from Box and adds it to the FlowFile's attributes. Tags box, fetch, metadata, storage Input Requirement REQUIRED Supports Sensitive Dynamic Properties false Properties • Box Client Service Controller Service used to obtain a Box API connection. Display Name Box Client Service Description Controller Service used to obtain a Box API connection. API Name box-client-service Service Interface org.apache.nifi.box.controllerservices.BoxClientService Service Implementations org.apache.nifi.box.controllerservices.DeveloperBoxClientService org.apache.nifi.box.controllerservices.JsonConfigBasedBoxClientService Expression Language Scope Not Supported Sensitive false Required true • File ID The ID of the File to fetch metadata for Display Name File ID Description The ID of the File to fetch metadata for API Name File ID Default Value ${box.id} Expression Language Scope Environment variables and FlowFile Attributes Sensitive false Required true Relationships Name | Description | 
 failure | A FlowFile will be routed here if fetching the file metadata fails. | 
 not.found | FlowFiles for which the specified Box file was not found. | 
 success | A FlowFile will be routed here after successfully fetching the file metadata. |

 box.id | The id of the file | 
 filename | The name of the file |

 box.path.folder.ids | A comma separated list of file path_collection IDs |

 box.created.at | The creation date of the file | 
 box.owner | The name of the file owner | 
 box.owner.id | The ID of the file owner | 
 box.owner.login | The login of the file owner | 
 box.description | The description of the file | 
 box.etag | The etag of the file | 
 box.sha1 | The SHA-1 hash of the file | 
 box.content.created.at | The date the content was created | 
 box.content.modified.at | The date the content was modified | 
 box.item.status | The status of the file (active, trashed, etc.) | 
 box.sequence_id | The sequence ID of the file | 
 box.parent.folder.id | The ID of the parent folder | 
 box.trashed.at | The date the file was trashed, if applicable | 
 box.purged.at | The date the file was purged, if applicable |
