Tags aggregate, aggregation, mongo Input Requirement ALLOWED Supports Sensitive Dynamic Properties false • Additional Details for RunMongoAggregation 2.5.0 RunMongoAggregation

 Description:

 This processor runs a MongoDB aggregation query based on user-defined settings. The following is an example of such a query (and what the expected input looks like):

 [
 {
 "$project": {
 "domain": 1
 },
 "$group": {
 "_id": {
 "domain": "$domain"
 },
 "total": {
 "$sum": 1
 }
 }
 }
]
