 Bundle org.apache.nifi | nifi-box-nar Description Fetches file metadata for each file in a Box Folder. Takes a flowFile with a folder ID attribute and outputs flowFiles with records containing all file metadata. Tags box, fetch, files, folder, storage Input Requirement REQUIRED Supports Sensitive Dynamic Properties false Properties • Box Client Service Controller Service used to obtain a Box API connection.
