 Bundle org.apache.nifi | nifi-standard-nar Description Counts various metrics on incoming text. The requested results will be recorded as attributes. The resulting flowfile will not have its content modified. Tags character, count, line, text, word Input Requirement REQUIRED Supports Sensitive Dynamic Properties false Properties • Call Immediate Adjustment If true, the counter will be updated immediately, without regard to whether the ProcessSession is commit or rolled back;otherwise, the counter will be incremented only if and when the ProcessSession is committed.
