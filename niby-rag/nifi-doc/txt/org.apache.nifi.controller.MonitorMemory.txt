 Bundle org.apache.nifi | nifi-standard-nar Description Checks the amount of Java Heap available in the JVM for a particular JVM Memory Pool. If the amount of space used exceeds some configurable threshold, will warn (via a log message and System-Level Bulletin) that the memory pool is exceeding this threshold. Tags garbage collection, gc, heap, jvm, memory, monitor, warning Input Requirement Supports Sensitive Dynamic Properties false Properties • Memory Pool The name of the JVM Memory Pool to monitor. The allowed values for Memory Pools are platform and JVM dependent and may vary for different versions of Java and from published documentation.
