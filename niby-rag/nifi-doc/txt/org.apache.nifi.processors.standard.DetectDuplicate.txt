 Bundle org.apache.nifi | nifi-standard-nar Description Caches a value, computed from FlowFile attributes, for each incoming FlowFile and determines if the cached value has already been seen. If so, routes the FlowFile to 'duplicate' with an attribute named 'original.identifier' that specifies the original FlowFile's "description", which is specified in the <FlowFile Description> property. If the FlowFile is not determined to be a duplicate, the Processor routes the FlowFile to 'non-duplicate' Tags dedupe, dupe, duplicate, hash Input Requirement REQUIRED Supports Sensitive Dynamic Properties false Properties • Age Off Duration Time interval to age off cached FlowFiles Display Name Age Off Duration Description Time interval to age off cached FlowFiles API Name Age Off Duration Expression Language Scope Not Supported Sensitive false Required false • Cache Entry Identifier A FlowFile attribute, or the results of an Attribute Expression Language statement, which will be evaluated against a FlowFile in order to determine the value used to identify duplicates; it is this value that is cached Display Name Cache Entry Identifier Description A FlowFile attribute, or the results of an Attribute Expression Language statement, which will be evaluated against a FlowFile in order to determine the value used to identify duplicates; it is this value that is cached API Name Cache Entry Identifier Default Value ${hash.value} Expression Language Scope Environment variables and FlowFile Attributes Sensitive false Required true • Cache The Entry Identifier When true this cause the processor to check for duplicates and cache the Entry Identifier. When false, the processor would only check for duplicates and not cache the Entry Identifier, requiring another processor to add identifiers to the distributed cache.
